Vefsíðugerð og útgáfustýring

Vefsíðugerð

Til að byrja með þurfti að finna sniðmát sem hentaði. Kröfurnar til sniðmátsins voru þær að það átti að vera stílhreint og minimalísktt en þó þannig að allt væri til alls. Fyrir valinu varð sniðmátið Massively frá HTML5 UP

Sniðmátið er bæði nokkuð minimalískt en er samt sem áður góð beinagrind með öllum þeim eiginleikum sem gerðar voru kröfur til, t.d. að vera með undirsíður o.s.frv. en það fylgir ekki einfaldari sniðmátum sem prófuð voru á undan.

Hvítur bakgrunnur, svart letur og bakgrunnsmynd þótti gefa stílhreint og faglegt útlit. Til þess að einfalda viðmótið var sú ákvörðun tekin að eftir stutta kynningu myndu verkefni annarinanr blasa við á forsíðunni þar sem heiti þeirra og stutt lýsing kemur fram og svo er hægt að nálgast ferilskrána í áberandi flipa að ofan.

Við forritun vefsíðunnar var sniðmátinu breytt og bætt þar til síðan uppfyllti hönnunarforsendurnar. Þar sem engin reynsla var til staðar í forritunartungumálinu HTML fór smá tími í að setja sig inn í það og hjálpaði að sækja sér upplýsingar á netinu ásamt að hljóta aðstoð frá gervigreindartólum þegar upp komu villumeldingar sem ekki tókst að greiða úr upp á eigin spýtur.

Útgáfa vefsíðunar

Til að byrja með útbjó ég aðgang inn á Git ásamt því að hlaða niður Git á tölvuna.

Til viðmiðunar nýttust myndbönd kennarans ágætlega við það að fá grófa mynd af ferlinu, svo fylgdi ég leiðbeiningum af síðunni Geeks for Geeks How to push anything to GitHub using Git Bash. Þær leiðbeiningar eru nokkuð góðar og er hægt að fylgja nokkuð blint.

Í grófum dráttum þarf að búa til nýtt "repository". Finna þarf svo möppuna sem stendur til að hlaða upp á git og velja "open Git Bash here". Git Bash opnast svo og setja þarf inn slóðina að geymslunni (e. repository). Næsta skref er að afrita allar skrár sem fylgja vefsíðunni yfir og tekur svo við staðfesting á því í Git Bash.

Það eina sem fylgir ekki leiðbeiningunum How to push anything to GitHub using Git Bash. Er að skrá þarf sig inn í miðju ferli en eftir smá vesen kom í ljós að leiðbeiningar fylgdu villumeldingunni og einfaldlega þarf að fara eftir þeim.

en eftir það þarf einfaldlega að slá inn "git comitt -m " Svo lýsingu á því sem breyta á" og svo að að ýta þessu yfir með git push origin main .

Í lokin þar svo að fara inn á Git > Settings > Pages og passa að eftirfarandi sé valið

Þá er vefsíðan birt.

Þegar breyta þarf vefsíðunni þarf að gera þær breytingar inn í möppunni sem ýtt var yfir á Github og svo hlaða þeim aftur upp með Git Bash.

Væntingar til áfangans

Væntingar mínar til áfangans eru þær að ég öðlist betri færni í því að taka hönnunarferlið alla leið, allt frá hugmynd í tilbúinn hlut og allt þar á milli. Ég hlakka sérstaklega til að læra á Fusion og 3d prentun en það virðist vera mjög hagnýtt. Ég er enn þá ekki búinn að ákveða lokaverkefni en ég vil helst gera eitthvað sem ég get svo nýtt mér.
